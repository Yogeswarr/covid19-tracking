About

Covid-19 INDIA, an web page which displays the COVID-19 stats of India as well as stats of states and union teritories. This app was built on Angular 12 framework, which is a open source used to build scalable single page wep application in typescript (TS). Have used SCSS for styling the components.

Have used D3 (Data Driven Documents) for the visualization of the country / state maps with the required stats in the form of a choropleth map. Click on any state or search to get the information(stats) of the particular state/union teritories. The bar-graph represents the number of vaccination in terms of 1st dose / fully vaccinated comparing it with the population of the country / state.

Data was fetched from an API using the HTTP-client inbuilt in angular core. The API link can be found here.
